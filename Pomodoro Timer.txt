Pomodoro Timer

Function:
-25 min intervals
-After every interval 3-5min break
-Every 4 interval 15-30min long break

psudeo coder 

1. on button click, start Timer
2. alert "lets get to work"
3. when timer reachers 0, ring sound 
4. alert "Take a break"
5. count first break.
6. start break timer.
7. when breakTimer reaches 0. play sound.
8. alert "lets get to work"
9. repeat steps 2-7 until break count reaches 4.
10. when breakCount reaches 4, then play different sound.
11. alert "take a long break"
12. start longBreakTimmer.
13. when longBreakTimmer reaches 0 then repeat all.
